# stock-market-simulator
Stock market simulator using Node.js and Yahoo Finance API

This project allows users to search companies worldwide and fetch live stock market data.

## Tech Stack
- Node.js
- Express.js
- Axios
- Yahoo Finance API

## Features
- Search global companies
- Fetch real-time stock data
- REST API backend

## Run Locally

1. Install dependencies
npm install

2. Start server
node server.js

Server runs on:
http://localhost:5000
